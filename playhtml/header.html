HTML5 <header> 元素详解

什么是<header>元素？

<header>是HTML5中引入的语义化标签，用于定义文档或文档部分的页眉（介绍性内容）。

提示： 与旧的<div class="header">相比，<header>提供了更好的语义含义，有助于SEO和可访问性。

<!-- 基本用法 --> <header> <h1>网站标题</h1> <p>网站标语或简介</p> </header>

<header>的常见用途

1. 页面顶部header

通常包含网站logo、导航菜单和搜索框等。

<header> <div class="logo">我的网站</div> <nav> <ul> <li><a href="#">首页</a></li> <li><a href="#">关于</a></li> <li><a href="#">联系</a></li> </ul> </nav> </header>

2. 文章header

每个<article>或<section>都可以有自己的<header>。

文章标题

作者: 张三 | 发布日期: 2023-05-15

这里是文章内容...

<article> <header> <h2>文章标题</h2> <p>作者: 张三 | 发布日期: 2023-05-15</p> </header> <p>这里是文章内容...</p> </article>

<header>与<head>的区别

初学者常混淆<header>和<head>：

  • <head>：包含文档的元数据（标题、字符集、样式表链接等），对用户不可见
  • <header>：包含可见的介绍性内容（标题、导航等）
<!DOCTYPE html> <html> <head> <!-- 元数据 --> <title>页面标题</title> <meta charset="UTF-8"> <link rel="stylesheet" href="styles.css"> </head> <body> <header> <!-- 可见的页眉内容 --> <h1>网站标题</h1> </header> </body> </html>

最佳实践

提示： 一个页面可以有多个<header>元素，但通常不应嵌套使用。

  1. 使用<header>包裹介绍性内容（标题、logo、导航等）
  2. 可以在<article>、<section>等元素内部使用<header>
  3. 避免在<footer>、<address>或另一个<header>内部使用<header>
  4. 通常包含一个或多个标题元素（<h1>-<h6>）
